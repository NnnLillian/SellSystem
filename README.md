# SellSystem

### 技术相关
前端：Vue.js构建的WebApp（部署在虚拟机中）

后端：SpringBoot、后端界面：BootStrap + Freemarker + JQuery


> 数据库：SpringBoot+JPA
>
> 缓存：Redis（分布式session与分布式锁）
>
> 消息推送：Websocket

前端与后端通过RestFul风格接口相连。
